## Models Implemented

### Classical ML Baselines
- K-Nearest Neighbors (KNN)
- Decision Tree
- HOG + SVM

- Custom CNN
- DenseNet121 (Transfer Learning)
- ResNet50 (Transfer Learning)
- EfficientNet-B0 (Transfer Learning)

## Datasets
- Shenzhen Hospital Chest X-ray Set (TB vs Normal)
